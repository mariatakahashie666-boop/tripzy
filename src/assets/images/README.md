# Background Image

Please add a background image named `travel-background.jpg` to this directory.

The image should show 2 people holding their passports in front of an airplane to match the travel theme.

Recommended specifications:
- Resolution: 1920x1080 or higher
- Format: JPG or PNG
- Subject: Travel/airport scene with people holding passports
- Style: Professional, bright, welcoming

The image will be used as a background for the light mode with a semi-transparent overlay to maintain readability.
